# Ongera Ubeho — "Live More · Choose Life"

Community-based Youth & Family Mentorship Hub in Rwanda, reconnecting struggling youth, street children, and worried parents with trained Kinyarwanda-speaking mentors.

## Website

The full website lives in [`website/`](website/):

| Page | Purpose |
|---|---|
| `index.html` | Home — mission, the challenge, the two tracks, commitments |
| `about.html` | Story, the crisis, values, team |
| `programs.html` | Family Mentorship (Track 1), Street Youth Rehabilitation (Track 2), platform, FAQ |
| `impact.html` | Pilot targets, transparent budget, roadmap, recognition |
| `get-involved.html` | Funding, institutional partnership, mentoring, volunteering |

Shared assets: `website/css/styles.css`, `website/js/main.js`, photography in `website/assets/img/` (see `website/assets/CREDITS.md` for Unsplash attribution).

**Deploying:** upload the entire `website/` folder as-is (e.g. GitHub Pages, Netlify, or any static host). The pages reference `assets/`, `css/`, and `js/` by relative path, so the folder must stay together.
